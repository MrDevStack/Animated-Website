# Morning to Night Scrolling Animated Website

A cool scrolling animated website where the scene changes beautifully from morning to night as the user scrolls.

## Features

- Smooth scrolling animation
- Morning to night transition
- Creative animated design
- Responsive layout
- Beginner-friendly project
- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
